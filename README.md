# Sistema Bancário em Java

Projeto desenvolvido como desafio do curso da [Alura](https://www.alura.com.br/).  
O objetivo foi criar um sistema bancário simples para praticar **Java**, lógica de programação e estruturas de decisão.

## Funcionalidades

- Consultar saldo da conta
- Receber valores (depósito)
- Transferir valores com validação de saldo
- Menu interativo em loop

## Tecnologias utilizadas

- Java 11+
- Scanner (para entrada de dados)
- Estruturas condicionais (`if`, `else`, `switch`)
- Loops (`while`)
